🔋 Chargeur de Batterie Solaire à Hacheur Abaisseur
📌 Contexte du projet

Ce projet a été réalisé dans le cadre de ma formation à l’Université Lyon 1 Claude Bernard (2021/2022). L’objectif était de concevoir un système de recharge efficace d’une batterie 12 V à partir d’un panneau solaire 18 V en utilisant un convertisseur abaisseur (buck) contrôlé par un Arduino.
⚙️ Principe de fonctionnement

Le système est basé sur un hacheur abaisseur (buck converter) commandé en PWM par un Arduino Uno.
🟡 Problème à résoudre :

La tension du panneau solaire n’est pas constante. Elle varie selon la luminosité ambiante (ensoleillement), ce qui rend la recharge directe inefficace, voire dangereuse pour la batterie.
🟢 Solution :

Le rapport cyclique (duty cycle) du signal PWM est ajusté dynamiquement par l’Arduino afin de stabiliser la tension de sortie à environ 13 V, valeur adaptée à la recharge d’une batterie 12 V.
🔁 Boucle de régulation

    Mesure de la tension de sortie via un pont diviseur résistif connecté à une entrée analogique de l’Arduino.

    Comparaison avec la consigne (par exemple 13.0 V).

    Ajustement du PWM : si la tension chute (ex. nuage), le duty cycle augmente pour compenser. Si la tension monte, il diminue.

    La boucle se répète toutes les 50 à 100 ms.

// Extrait simplifié du code Arduino
float Vout;        // Tension de sortie mesurée
float consigne = 13.0; // Tension cible (V)
int pwm;           // Valeur PWM (0-255)

void loop() {
  int analogValue = analogRead(A0); // Lecture pont diviseur
  Vout = analogValue * (5.0 / 1023.0) * coef_diviseur;
  if (Vout < consigne) pwm += 1;
  else if (Vout > consigne) pwm -= 1;
  pwm = constrain(pwm, 0, 255);
  analogWrite(9, pwm); // Commande MOSFET
  delay(100);
}

🔌 Schéma de principe

    Entrée : Panneau solaire 18 V

    Sortie : Batterie 12 V

    Composants principaux :

        MOSFET de puissance

        Diode de roue libre

        Inductance de stockage

        Condensateur de filtrage

        Arduino Uno (PWM + mesure analogique)

🧪 Résultats

    Tension de sortie régulée à ~13 V malgré les variations de la lumière du jour.

    Recharge efficace et sécurisée sans surchauffe ni surcharge.

    Consommation optimisée selon les conditions réelles.

🛠️ Environnement technique

    MOSFET de puissance

    Diode de roue libre rapide

    Convertisseur Buck

    Arduino Uno

    Panneau solaire 18 V

    Batterie 12 V (plomb ou LiFePO4)
